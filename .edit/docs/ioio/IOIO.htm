IOIO

The IOIO object is used to manage the connection between your Android phone and the IOIO board. Usually you create one these objects at the start of you program like this:

 ioio = app.CreateIOIO();

See the CreateIOIO function for more info.

The following functions are provided by the IOIO object:

  • CreateDigitalOutput
  • CreateDigitalInput
  • CreatePwmOutput
  • CreateAnalogInput
  • CreateUart
